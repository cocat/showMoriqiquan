# 第一部分：亚马逊假发 Listing 图片解剖

## 1.1 一个标准假发 Listing 需要哪些图？

头部卖家（Luvme / UNice / ISEE）的 listing 通常有 **7-9 张图**，结构如下：

```
图片位  │  类型            │  内容                        │  你能用 AI 做吗？
────────┼─────────────────┼─────────────────────────────┼──────────────
图 1    │  白底主图        │  模特正面佩戴，纯白底          │  ✅ 核心产品
图 2    │  多角度展示      │  正面/侧面/背面 三合一拼图     │  ✅ 核心产品
图 3    │  细节特写        │  发际线/蕾丝/头皮效果特写      │  ⚠️ 需要真实产品图
图 4    │  信息图          │  参数标注（长度/密度/帽型）     │  ✅ AI 图+文字排版
图 5    │  场景图 1        │  室内/办公室/咖啡厅生活照      │  ✅ 核心产品
图 6    │  场景图 2        │  户外/街拍/社交场合            │  ✅ 核心产品
图 7    │  对比/色卡图     │  不同光线下的色泽对比          │  ⚠️ 需要搭配真图
图 8    │  品牌故事        │  品牌理念/售后承诺横幅         │  ✅ 设计+AI 图
图 9    │  视频封面        │  动态展示缩略图               │  ✅
```

> [!IMPORTANT]
> **你的服务能覆盖 7-9 张中的 5-6 张**（图 1/2/4/5/6/8）。图 3（发际线特写）和图 7（色泽对比）需要真实产品照片，这是 AI 的短板。**但 5-6 张已经占了整套图的 70%，价值足够大。**

## 1.2 每张图的详细要求

### 图 1：白底主图（最重要，决定点击率）

```
亚马逊硬性要求：
  • 纯白背景 RGB(255,255,255)
  • 产品占画面 85% 以上
  • 不可有文字/水印/logo/配件
  • 必须是产品实际外观
  • 分辨率 ≥ 1600px（开启缩放）

你出图的标准：
  • 模特正面，面部清晰
  • 假发占据画面主体
  • 自然光效果，无硬阴影
  • 发丝纹理清晰可见
  • 模特表情自然，微笑或平静

注意：亚马逊说"不可包含模特"但假发品类是例外
——假发必须展示佩戴效果，允许使用模特
```

### 图 2：多角度拼图

```
布局：3 张拼成一张图
  ┌──────────────────────────┐
  │  [正面]  [侧面]  [背面]   │
  │                          │
  │   45°左   正面    45°右   │
  └──────────────────────────┘

你需要出：同一模特、同一假发、3 个角度
技术重点：保持模特面部和假发在三个角度的一致性
```

### 图 5-6：生活场景图（高转化率关键）

```
场景类型按热门程度排序：
  1. 室内自然光（窗边、沙发旁）       → 温暖亲切
  2. 城市街拍（人行道、咖啡厅外）      → 时尚感
  3. 办公室/职场（桌前、会议室）       → 职业感
  4. 晚宴/社交（酒店大堂、餐厅）      → 高端感
  5. 户外运动（公园、海边）            → 活力感

你需要做的：
  • 为每个场景准备 2-3 套背景模板
  • 模特需变换姿态（站/坐/回头/走路）
  • 光照需与场景匹配（室内暖光/户外自然光）
```

# 第三部分：你需要出什么图

## 3.1 按假发类型的出图标准

### Straight（直发）— 最简单，先用来练手

```
出图要求：
  • 发丝必须顺直，无杂乱
  • 光泽感要强（直发的卖点就是顺滑光泽）
  • 发尾收束自然，不散乱

必出角度：
  ├── 正面（展示刘海和分缝）
  ├── 侧面 45°（展示长度和顺滑度）
  ├── 背面（展示发尾和整体长度）
  └── 半侧面回头（展示发丝流动感）

场景搭配：
  └── 职场/办公室风格最匹配（干练利落感）
```

### Body Wave（大波浪）— 销量最大

```
出图要求：
  • 波浪弧度要自然均匀
  • 发丝要有蓬松感（体积感）
  • 光影要体现波浪的层次感

必出角度：
  ├── 正面（展示波浪从上到下的走向）
  ├── 侧面（展示波浪的立体感/厚度）
  ├── 背面（展示整体波浪效果）
  ├── 风吹效果（展示发丝动态飘逸感）
  └── 特写（展示波浪纹理细节）

场景搭配：
  └── 街拍/休闲场景最匹配（时尚活力感）
```

### Lace Front（前蕾丝）— 客单价最高

```
出图要求：
  • 发际线必须自然（这是客户最关注的）
  • 蕾丝与皮肤的过渡要无痕
  • 分缝线（Part Line）要逼真

必出角度：
  ├── 正面（重点展示发际线）
  ├── 额头特写（展示蕾丝与肤色融合）
  ├── 露额角度（侧分/中分展示分缝效果）
  └── 生活场景正常距离（证明远看也自然）

⚠️ AI 难点：发际线与皮肤的过渡是最考验技术的部分
→ 这里可能需要额外的 Inpainting 修复步骤
```

## 3.2 你需要为每款假发交付的标准图片集

```
标准套餐（5 张/款）：
  ├── 图 1：白底正面佩戴图（用于亚马逊主图）
  ├── 图 2：白底侧面 45° 图
  ├── 图 3：白底背面图
  ├── 图 4：场景图 1（室内自然光）
  └── 图 5：场景图 2（户外/街拍）

升级套餐（8 张/款）：
  ├── 标准 5 张
  ├── 图 6：不同肤色模特正面对比（深棕 vs 浅棕）
  ├── 图 7：信息图模板（长度/密度标注底图）
  └── 图 8：A+ Content 横幅图
```

# 第六部分：对比图/模特图的制作方法

## 6.1 前后对比图制作标准

```
工具：Canva / Figma / PS 均可

布局方案 A：左右对比（最直观）
  ┌──────────────────────────────┐
  │    BEFORE      →     AFTER   │
  │  ┌─────────┐    ┌─────────┐ │
  │  │假人头图  │    │模特佩戴  │ │
  │  │         │    │  图     │ │
  │  └─────────┘    └─────────┘ │
  │      原始素材         AI 生成  │
  └──────────────────────────────┘

布局方案 B：上下对比（适合竖屏手机）
  ┌─────────────────┐
  │   [假人头原图]     │
  │    BEFORE ↓       │
  ├─────────────────┤
  │   [模特佩戴图]     │
  │    AFTER ✨       │
  └─────────────────┘

布局方案 C：一拖多（展示多肤色能力）
  ┌──────────────────────────────────┐
  │  [原图]  →  [深棕] [浅棕] [白人]  │
  │  1 张底图生成 3 个肤色的模特图      │
  └──────────────────────────────────┘

制作要求：
  • 左边原图保留原始的"粗糙感"（不要美化）
  • 右边 AI 图展示最佳效果
  • 对比越强烈，传播力越强
  • 添加简洁的文字标注
  • 底部加品牌 logo 或联系方式
  • 尺寸：1080×1350（竖版，小红书/IG 最佳）
```

## 6.2 模特图展示的制作规范

```
用于展示的模特图需要比交付图更精致：

展示用模特图标准：
  ├── 分辨率：至少 2K（2048×2048）
  ├── 模特：选最好看的面孔
  ├── 假发：选纹理最清晰的品类（Body Wave / Straight）
  ├── 场景：选最有质感的（杂志风 > 白底棚拍）
  ├── 后处理：额外加一层调色（提升高级感）
  └── 水印：角落加半透明品牌标识

展示图组合套装（用于各个推广渠道）：
  ├── 3 组前后对比图（不同品类：直发/波浪/卷发）
  ├── 1 组多肤色展示（1 顶假发 × 4 个肤色）
  ├── 1 组多场景展示（1 顶假发 × 4 个场景）
  ├── 1 张全景拼图（9宫格，展示整体实力）
  └── 1 段 15 秒视频（原图→AI 图转场动画）
```

# 第八部分：补充素材（漏洞修订 + 风险补强）

> [!IMPORTANT]
> 以下补充内容用于修正原手册里最容易导致“能出图，但难成交、难续费、难规模化”的风险点。建议保留原手册的行动性，同时把这里的边界、SOP 和合规提示并入实际执行。

## 8.1 这份手册最需要修正的 10 个点

### 1. 不要把“AI 模特白底图”默认写成亚马逊主图方案

```
更稳的表述：
  • 主图优先：真实产品图 / 真实佩戴图 / 基于真实图的合规修图
  • 副图优先：AI 场景图 / 信息图 / 多肤色展示 / 视频封面
  • 纯 AI 生成主图：只能作为测试项，不能默认承诺“可稳定上架”

原因：
  • Amazon 主图规则强调：actual product / no illustration
  • Amazon 对“允许模特”的公开例外表述主要绑定 apparel / apparel accessory
  • 假发是否能被系统或审核员稳定视作这一例外，不能当作低风险前提
```

### 2. 你的核心卖点要从“AI 出图”改成“真实素材增强 + 上新提效”

```
建议改口径：
  • 我们不是凭空造图
  • 我们是基于真实产品素材，做更统一、更快、更便宜的电商视觉生产

这样更容易解决三个问题：
  • 客户不担心“和实物不符”
  • 你更容易解释为什么需要素材标准
  • 后续更容易接品牌统一、A/B 测试、视频等高客单业务
```

### 3. 缺少客户素材输入标准，后期返工风险极高

```
每个 SKU 至少要求客户提供：
  • 正面、左侧、右侧、背面图
  • 头顶分缝图
  • 发际线 close-up
  • lace close-up
  • 长度尺对照图
  • 色卡或自然光色泽图
  • 包装/配件图（如需展示）

补一句：
  “素材不达标时，只承诺出创意参考图，不承诺可直接上架。”
```

### 4. 你现在的标准套餐偏“便宜”，但不够“结果导向”

```
原套餐问题：
  • 只有 5 张图
  • 没有视频
  • 没有明确主图是否包含在交付承诺里

建议调整：
  • 标准版：6 张图
  • 或者：5 张图 + 1 个视频封面 / 15 秒转场视频
  • 在销售页明确区分：可上架素材 vs 展示案例素材
```

### 5. A+ Content 不能默认卖给所有商家

```
先问客户两个问题：
  • 是否完成 Brand Registry
  • 是否用品牌店铺/品牌 ASIN 在运营

如果没有品牌备案：
  • 不卖 A+ 正式交付
  • 可以卖 A+ 结构稿 / 横幅草图 / 品牌视觉方案
```

### 6. 价格和免费试单策略过低，容易把自己锁进低利润

```
原风险：
  • 80-100 元 / 款
  • 免费试 2-3 款
  • 48 小时出图

更稳的做法：
  • 免费只做“审图建议”或“1 张低水印 demo”
  • 真正试单改成低价付费 pilot
  • 交付默认 1 次修改，超出单独收费
  • 加急、补图、补肤色、补场景全部单列价格
```

### 7. 用“网上找的假发图”做案例，有版权和信任风险

```
原 Week 1 写法建议改掉：
  • 不要直接用未知来源的品牌图做商业案例
  • 更稳的是：
    1. 自购 2-3 顶样品
    2. 找可授权工厂拿样
    3. 用明确可商用素材

否则客户会问：
  “这个案例到底是不是你们自己的？”
```

### 8. 缺少 ICP 筛选机制，容易把时间花在低质量线索上

```
建议每个线索都打分（1-5 分）：
  • 是否有 Brand Registry
  • 是否已有 A+ / 视频
  • SKU 数量是否 > 20
  • 当前图片质量是否一般但不算烂
  • 能否找到老板 / 运营负责人 / 美工负责人

优先联系总分最高的 10 家，不要平均发力
```

### 9. 小红书内容方向偏“震撼流量”，缺少 B2B 信任设计

```
建议多发这几类内容：
  • 同一 SKU 统一全套视觉前后对比
  • 怎样降低返工率
  • 怎样让不同长度/颜色维持同一品牌风格
  • 用什么素材最容易出可用图
  • 哪些 AI 图适合副图，不建议碰主图

少发：
  • “逆天”“炸了”“你分得出来吗”这类纯情绪标题
```

### 10. 文档少了“验收标准”和“数据闭环”

```
必须新增两个模块：
  • QA 验收清单
  • 效果追踪表

如果没有数据闭环：
  • 你永远只能卖“便宜和快”
  • 很难升级到“提升点击率 / 提升转化率 / 提升上新效率”
```

## 8.2 建议新增的资格预审问题

```
接触客户后，先问这 8 个问题：
  1. 你们主做 Amazon 还是独立站 / TikTok Shop？
  2. 目前一个月上新多少个 SKU？
  3. 现在图片是谁在做？内部美工还是外包？
  4. 有没有固定模特或摄影团队？
  5. 有没有 Brand Registry？
  6. 你们最想解决的是主图点击、详情转化，还是上新效率？
  7. 你们能不能提供标准化的产品拍摄素材？
  8. 如果先做 1-3 款试单，预算大概在哪个区间？
```

## 8.3 客户素材输入 SOP（建议直接复制到接单流程）

```
每个 SKU 需提交：
  □ 正面图 1 张
  □ 左侧 45° 1 张
  □ 右侧 45° 1 张
  □ 背面图 1 张
  □ 头顶分缝图 1 张
  □ 发际线 close-up 1 张
  □ lace close-up 1 张
  □ 长度尺图 1 张
  □ 自然光色泽图 1 张
  □ 如需展示包装：包装图 1-2 张

拍摄要求：
  • 背景尽量纯净
  • 光线均匀，不要黄光
  • 同一顶假发拍摄距离尽量一致
  • 不要开过度美颜
  • 原图直接发，不要二次压缩
```

## 8.4 交付 QA 验收清单

```
每张图交付前检查：
  □ 发量和原产品是否一致
  □ 长度是否一致
  □ 分缝位置是否一致
  □ 发际线是否过度假
  □ lace 与皮肤衔接是否自然
  □ 颜色是否偏离原图
  □ 发尾是否异常断裂/糊掉
  □ 耳朵、手指、肩颈是否正常
  □ 光照方向是否统一
  □ 服装和场景是否喧宾夺主
  □ 文案参数是否与产品一致
  □ 是否区分“案例图”和“可上架图”
```

## 8.5 建议重写的产品结构

```
产品 1：审图诊断
  • 作用：先把客户问题讲清楚，建立专业度
  • 交付：1 份问题诊断 + 1 页改造建议

产品 2：单 SKU 付费试单
  • 作用：低风险成交
  • 交付：1 款 4-6 张图

产品 3：批量上新包
  • 作用：做利润
  • 交付：5-20 款统一风格图

产品 4：品牌视觉统一包
  • 作用：做高客单
  • 交付：品牌模特设定 + 视觉模板 + A+ 草图 + 店铺横幅

产品 5：月度托管
  • 作用：做复购
  • 交付：每月固定款数、固定场景、固定周转时效
```

## 8.6 合规边界（截至 2026-04-03）

```
Amazon 侧：
  • 主图规则强调：actual product、纯白背景、无水印、无插画
  • Amazon 公开信息里，对“模特可出现在主图”的低风险例外，主要见于 apparel / apparel accessory
  • 所以假发类主图不要默认承诺“纯 AI 模特图可稳定通过”
  • Amazon 公开建议：至少 6 张图 + 1 个视频
  • A+ Content 需先确认客户是否具备 Brand Registry 资格

中国内容侧：
  • 《人工智能生成合成内容标识办法》已于 2025-09-01 生效
  • 你在小红书、视频号、案例海报里展示 AI 生成内容时，要考虑显式标识和用户声明
  • 不建议用“真人实拍”误导表述去包装纯 AI 内容
```

## 8.7 你下一版文档必须加上的 7 个章节

```
  1. 服务边界：哪些内容可安全交付，哪些只能做测试素材
  2. 客户准入：谁是优先客户，谁先不碰
  3. 输入 SOP：客户必须提供什么
  4. QA 验收：交付前怎么检查
  5. 报价规则：修改次数、加急费、案例授权、保密条款
  6. 数据闭环：CTR、CVR、sessions、退货率怎么追
  7. 案例授权：哪些图能公开发小红书，哪些只能内部展示
```

## 8.8 建议保留的原文优点

```
这份原手册有三个明显优点，不建议丢：
  • 动作感强：读完知道下一步干什么
  • 产业带理解对：许昌 / 青岛是有效切入口
  • 内容营销意识强：知道要做案例、对比图、微信私域

更好的做法不是推翻它，而是把它升级成：
  “有边界、有 SOP、有报价逻辑、有数据闭环”的版本
```
